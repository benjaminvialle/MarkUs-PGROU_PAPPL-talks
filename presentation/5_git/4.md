Git (5/5)
================

Change state
---------------
$ `git checkout 82f5`

$ `git checkout master`

Commit everything
------------------
$ `git commit -a`

Clone a repository
-------------------
$ `git clone git://server/path/to/files`

What changed?
--------------
$ `git diff`

$ `git diff 1b6d "master~2"`

$ `git whatchanged --since="2 weeks ago"`
